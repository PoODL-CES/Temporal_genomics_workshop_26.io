Temporal Genomics Workshop

Exploring Ancient DNA, Evolution, and Bioinformatics

About the Workshop

This workshop introduces participants to a reproducible pipeline which focuses on ancient DNA analysis, sequencing workflows, and evolutionary insights using modern bioinformatics tools.

📚 What You'll Learn

  • Processing raw FASTQ data
  • Ancient DNA challenges & damage patterns
  • Alignment using BWA
  • Variant calling & filtering
  • Population genomics analysis
